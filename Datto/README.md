# Datto

Scripts for managing Datto SaaS Protection and Datto Endpoint Backup for PCs. Run interactively from a technician workstation or deployed via NinjaOne.

See [HOWTO.md](../HOWTO.md) for guidance on downloading and running scripts.

---

## Scripts

| Script | Description |
|---|---|
| [SaaS Protection Bulk Seat Change.ps1](SaaS%20Protection%20Bulk%20Seat%20Change.ps1) | PowerShell version — bulk-modifies seat assignments in Datto SaaS Protection across one or more accounts |
| [SaaS Protection Bulk Seat Change.py](SaaS%20Protection%20Bulk%20Seat%20Change.py) | Python version of the same bulk seat change operation |
| [Uninstall Datto Endpoint Backup.ps1](Uninstall%20Datto%20Endpoint%20Backup.ps1) | Silently uninstalls the Datto Endpoint Backup for PCs agent from a Windows device |
